# Express Note Taker   
[![License: MIT](https://img.shields.io/badge/License-MIT-green)](https://opensource.org/licenses/MIT)

## Description
This is an application run on Express and made, in-part by someone run on espresso. This app lets users input and save notes and gives the option to delete those notes as well.

## Installation
You can visit the deployed application link, or with the proper npm packages run it through your command line interface.

## Usage
User types the title and note text in the designated areas, a clickable save icon appears in the top right corner, if a note is no longer needed one can hit the trash can icon to delete it.
